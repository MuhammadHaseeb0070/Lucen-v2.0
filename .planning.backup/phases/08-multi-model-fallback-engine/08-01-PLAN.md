Implement the core sequential fallback execution engine (Primary -> Secondary -> Tertiary) for both streaming and non-streaming OpenRouter requests.

Output: Fallback logic in get-model-config, non-streaming fallback loop in chat-proxy, and streaming fallback loop in streamHandler.
</objective>
