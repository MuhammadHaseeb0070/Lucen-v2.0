# Requirements: Lucen v2.5 Web Search Optimization Milestone

**Defined:** 2026-06-09
**Core Value:** Secure, parallel, token-efficient, and premium web search that resolves user queries with accuracy and resilience.

## v1 Requirements

Requirements completed in this milestone:

### Web Search Optimization

- [x] **REQ-01**: Support parallelizable tool execution for `web_search` to fetch results concurrently.
- [x] **REQ-02**: Prevent step ceiling truncation errors by scaling `maxRounds` dynamically (up to 5 rounds for attachment + web search calls).
- [x] **REQ-03**: Prevent LLM tool leakages in final rounds by prompting with strong negative constraints when limits are reached.
- [x] **REQ-04**: Harden client-side XML tag stripping to defensively clean all possible parameters and leaked tool tags.
- [x] **REQ-05**: Restyle tool step statuses and domain citations to be beautiful, glassmorphic, and dynamic.
