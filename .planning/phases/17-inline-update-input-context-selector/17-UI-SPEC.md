## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | transparent / backdrop | Container background with backdrop-filter: blur(12px) |
| Secondary (30%) | var(--bg-surface) / var(--bg-muted) | Input field backing, inactive pills |
| Accent (10%) | var(--accent) / var(--accent-soft) | Focus borders, active pills, send button |
| Destructive | var(--danger) | Border flash and text on patch execution failure |

Accent reserved for: Active context pills (e.g. `1`, `2`, `3`), send button background (active state), and input border glow on focus.
